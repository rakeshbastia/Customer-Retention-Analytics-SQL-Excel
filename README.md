# Customer Retention Analytics Model - SQL & Excel

## Project Overview
Analyzed 50,000+ customer records using advanced SQL queries to identify retention patterns and customer behavior trends. Built comprehensive statistical analysis and interactive visualizations in Excel highlighting key retention metrics for strategic business insights and decision-making.

## Key Features
- **Large Dataset Analysis**: Processing and analyzing 50,000+ customer transaction records
- **Pattern Recognition**: SQL-based clustering to identify customer retention patterns
- **Behavioral Analysis**: Extraction of behavior data, purchase history, and engagement metrics
- **Statistical Analysis**: Advanced statistical methods to uncover retention drivers
- **Interactive Visualizations**: Excel dashboards with charts, pivot tables, and trend analysis
- **Segment Analysis**: Customer segmentation based on retention likelihood
- **Predictive Insights**: Identification of at-risk customers for proactive retention

## Project Deliverables
- **SQL Query Scripts**: Optimized queries for data extraction and transformation
- **Data Processing**: Cleaning and preparation of large datasets
- **Excel Models**: Statistical analysis workbooks with multiple sheets
- **Visualizations**: Charts, pivot tables, and interactive dashboards
- **Executive Summary**: Key findings and actionable recommendations

## Technical Stack
- **Database**: SQL Server / MySQL
- **Query Language**: SQL (CTEs, Window Functions, Aggregations)
- **Analysis Tool**: Microsoft Excel (Advanced Functions, Pivot Tables, Analysis ToolPak)
- **Visualization**: Excel Charts, Conditional Formatting
- **Data Processing**: VLOOKUP, INDEX-MATCH, Data Validation

## Analytical Methods
- **Cohort Analysis**: Tracking customer cohorts over time
- **Churn Prediction**: Identifying high-risk customer segments
- **RFM Analysis**: Recency-Frequency-Monetary value segmentation
- **Correlation Analysis**: Understanding factors influencing retention
- **Trend Analysis**: Historical patterns and seasonality detection

## Data Insights
- **Customer Segments**: Identified 5 key customer segments with different retention profiles
- **Retention Drivers**: Analyzed 15+ factors influencing customer retention
- **Risk Groups**: Flagged high-risk customer segments for targeted interventions
- **Behavioral Patterns**: Discovered key purchase patterns and engagement metrics

## Business Impact
